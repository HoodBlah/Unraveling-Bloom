# GET THE TEXTURE FROM TOM'S STORAGE

## Direct Link

Visit this GitHub path and download the PNG:
**https://github.com/tom5454/Toms-Storage/blob/main/Platform/Common/src/main/resources/assets/toms_storage/textures/gui/storage_terminal.png**

Click "Download" button on the right side.

## Where to Put It

Save as:  
`src/main/resources/assets/gardenoverflow/textures/gui/corporea_terminal.png`

## If You Can't Get It  

The GUI will automatically fall back to the vanilla chest texture. It will work fine, just won't look exactly like Tom's terminal.
